## Setup

Quick setup guide for the application.

1. Clone Repo

```bash
git clone git@github.com:gian-gg/TIPR.git
```

2. Install dependencies:

```bash
npm install
```

1. Build the project:

```bash
npm run build
```

4. Run with PM2 (and persist):

```bash
npx pm2 start app.js --name tipr
npx pm2 save
```
